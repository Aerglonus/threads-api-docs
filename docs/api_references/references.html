📒 API References

These are all the methods that can be used.

Important

Some of these methods require the following:

  • Profile must be public in order to retrieve threads,replies, profile info.
  • 2FA must be disabled (this is required to be able to login)
Advice

It is recommended to use token and userID instead of username and password and pass deviceID and device identifiers to avoid issues with Instagram.

📄️ ThreadsAPI Client

Client method to connect to threads.

📄️ login

Loging to Threads and returns `token` and `userID`

📄️ getToken

Retrieves the authenticated user bearer token

📄️ getCurrentUserID

Retrieves current username userID

📄️ getUserIDfromUsername

Retrieves userID from username

📄️ getIDfromInstagramUsername

Retrieves userID from Instagram username.

📄️ publish

Make a post to Threads

📄️ follow

Follow a given user on Threads

📄️ unfollow

Unfollow given user on Threads

📄️ like

Like a thread

📄️ unlike

Unlike a post in Threads

📄️ delete

Deletes a post from Threads

📄️ repost

Repost a post from Threads.

📄️ unrepost

Unrepost a post from Threads.

📄️ getThreadLikers

Get users that liked a given post

📄️ getPostIDFromThreadID

Generate postID from a ThreadID

📄️ getPostIDfromURL

Get the postID from a given URL

📄️ getThreads

Get replies from a given Thread

📄️ getTimeline

Get authenticated user timeline.

📄️ getUserProfileReplies

Get given user replies threads.

📄️ getUserProfileRepliesLoggedIn

Retrives given user replies.

📄️ getUserFollowers

Retrieves user follower list.

📄️ getUserFollowings

Retrieves user following list.

📄️ getUserProfile

Retrieves user data.

📄️ getUserProfileThreads

User threads without authentication.

📄️ getUserProfileThreadsLoggedIn

User threads with Authentication
