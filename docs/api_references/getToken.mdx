Retrieves the authenticated user bearer token

```typescript
const token = await threadsAPI.getToken();
```

| **requires** | **Type** | **Default** | **Description**       |
| ------------ | -------- | ----------- | --------------------- |
| `username`   | `string` | `undefined` | Threads username      |
| `password`   | `string` | `undefined` | Threads user password |
