---

Retrieves userID from Instagram username. `getUserIDfromUsername` fallsback to this.

```typescript
const userID = await threadsAPI.getUserIDfromInstagramUsername("username");
```

| **requires** | **Type** | **Default** | **Description**    |
| ------------ | -------- | ----------- | ------------------ |
| `username`   | `string` | `undefined` | Instagram username |
