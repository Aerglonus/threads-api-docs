# Getting Started

:::info Note
Keep in mind this is a Reverse-Engineered Unofficial API so some things might break as Meta updates the app.
:::

## Installation

To add Threads-api to your project, just can do the following:

<Tabs>
<TabItem value="js" label="yarn">

```bash
yarn add threads-api
```

</TabItem>
<TabItem value="py" label="pnpm">

```bash
pnpm install threads-api
```

</TabItem>
<TabItem value="java" label="npm">

```bash
npm install threads-api
```

</TabItem>
<TabItem value="deno" label="Deno ðŸ¦–">

```js
import ThreadsAPI from "npm:threads-api";
```

</TabItem>

</Tabs>

## Importing the library

Using typescript/ES6

```typescript
import { ThreadsAPI } from "threads-api";
```

Using Vanilla javascript:

```javascript
const ThreadsAPI = require("threads-api").ThreadsAPI;
```

```javascript
import ThreadsAPI from "npm:threads-api";
```
